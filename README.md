# SUIT TEST 02 - Lost Password

O objetivo dos testes presentes neste repositório é validar o funcionamento do fluxo de recuperação de senha.
As variáveis username e userEmail são obrigatórias para o teste.

# CT01
O objetivo do teste é validar o comportamento do sistema ao tentar recuperar uma senha sem informar os dados necessários

# CT02 
O objetivo do teste é validar o comportamento do sistema ao tentar recuperar uma senha informando dados incorretos.

# CT03
O objetivo do teste é validar o comportamento do sistema ao tentar recuperar uma senha informando dados corretos.

# Pré-requisitos

Plug-ins Selenium WebDriver.
Plug-ins NUnit.
